DUT Informatique : Projet Web

WEB :

La création et la conception de site web ou web design est la conception de l'interface web : l’architecture interactionnelle, l’organisation des pages, l’arborescence et la navigation dans un site web. La conception d'un design web tient compte des contraintes spécifiques du support Internet, notamment en termes d’ergonomie, d’utilisabilité et d’accessibilité.

Un site web peut être constitué par une simple page statique au format HTML éventuellement mise en forme ou enrichie de CSS incluant des images et des liens vers d'autres contenus. D'autres langages tels que le JavaScript ou le PHP peuvent le rendre plus intéractif mais ces derniers ne sont pas aborder lors de la 1ère Année de DUT Informatique.

Projet 2021 :

Le travail demandé était d'effectuer une étude des besoins du sujet choisi et d'implanter le site en utilisant les langages HTML 5 et CSS 3. L'organisation et la structure des pages étaient totalement libres cependant le site devaient comporter les différents éléments vues en cours. Une version mobile du site était demandé. Les élèves devaient travailler par groupe de 4 à 5 personnes et se répartir le travail.

Choix du sujet :

  • Sujet 1 : Création d'un site sur la protection de l'environnement

    L'idée de ce projet est de créer un site présentant différents aspects de la protection de l'environnement : pollution, changement climatique, développement durable, énergies renouvelables, alimention durable, etc...

    Le site doit présenter des informations pertinentes et intéressantes sur ces différents aspects.

  • Sujet 2 : Création d'un site web présentant l'univers du chocolat

    L'idée de ce projet est de créer un site pour présenter l'univers du chocolat.

    Ce site doit retracer l'histoire du chocolat et donner des informations pratiques sur le chocolat telles que le processus de fabrication, la composition des différents chocolats, etc...

  • Sujet 3 : Création d'un site web présentant l'univers des jeux vidéos

    L'idée de ce projet est de créer un site rassemblant des informations sur les jeux vidéo.

    Le site doit présenter un historique des jeux vidéo, en relation avec le développement des technologies. Il doit également proposer des informations pertinentes, intéressantes et variées sur les différents types de jeux vidéo.

    Il proposera également une partie sur les métiers des jeux vidéo en rapport avec l'informatique ainsi qu'une partie sur l'e-sport.
